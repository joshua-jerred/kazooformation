Kazooformation Update and Open Questions

Update

The big news, I attempted the transition from a discrete problem to a semi-continuous problem. It works! (somewhat).

  • Pulse Audio reading and writing is functional
  • 3-5x FFTs done across a single symbol, allowing for symbol edge detection
  • Byte alignment was done by creating a simple frame/packet structure, if the packet contents are restricted to 7-bit ASCII (AX.25 does this, for other reasons) then frame start/end can be unique.
  • Up to 4x symbols, already struggling with differentiation.
